- Build In SuperUser

- Customizable Power Menu
- Clear all Recent App's Button
- App opps
- Pulse Notification Light
- Customizable Battery light
- Ascending ring-tone
- Heads Up
- CM11 Theme Chooser

Lollipop Features

- Build With Linaro toolchain with -03 optimizations, For ROM and Kernel Compilations
- Build In SuperUser (Not Yet)
- Quick Notification Drawer
- Clear all Recent App's Button
- App opps

- Ascending ringtone

- Smooth scrolling
- Blacklist support
- Protected apps
- ProximityWake
- CM Profiles
- Volume rocker control
- Development animation transition mode 0.75 and 0.25
- Lockscreen shortcuts
- Weather in status bar
- OwnSettings
- Battery percentage
